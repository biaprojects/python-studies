# TUPLA
As tuplas são parecidas com uma lista, porém, elas são _imutáveis_.  

Sintaxe:  
```
nome = 'elemento1', 'elemento2'
```

Ou...  
```
nome = ("elemento1", "elemento2")
```

Os parênteses não são obrigatórios, mas geralmente colocamos.  

Exemplo:  
```
dias_uteis = ("segunda", "terca", "quarta", "quinta", "sexta")
```

## Tupla com apenas 1 elemento, como declarar?
Inclui uma vírgula depois do elemento.
