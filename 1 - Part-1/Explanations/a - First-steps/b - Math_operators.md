# Operadores matemáticos

|           Operação             | No programa | 
| ------------------------------ | ----------- |
|             Soma               |    a + b    |
|           Subtração            |    a - b    | 
|         Multiplicação          |    a * b    |
|         Exponenciação          |    a ** b   |
|            Divisão             |    a / b    |
| Inteiro da divisão (quociente) |    a // b   |
|   Resto da divisão - módulo    |    a % b    |

# Operadores de comparação

|    Operação    | No programa | 
| -------------- | ----------- |
|     maior      |      >      |
|     menor      |      <      |
| maior ou igual |     >=      |
| menor ou igual |     <=      |
|     igual      |     ==      |
|   diferente    |     !=      |
