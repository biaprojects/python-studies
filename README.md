## About
This repository started as a way to practice Git and GitHub while documenting my first steps in programming using Python.  
As time went by, it naturally grew into a broader collection of my learning journey. It brings together different experiences, including:  
- Self-study notes  
- Course content (USP – *Meninas Programadoras*)  
- Internship learnings  
- University notes (UNIVESP)  

Even though it includes content from different moments and contexts, it reflects my real progress and evolution in programming.  
This repository is continuously evolving as I learn new concepts and improve my coding practices.
